**Route Handler implementation:**
1. Import `createClient` from `@/lib/supabase/server` (same pattern as Server Actions).
2. Validate user via `supabase.auth.getUser()` — return 401 if not authenticated.
3. Validate entity param — return 400 if not one of the three allowed values.
4. Fetch all non-deleted records for the entity using Supabase queries:
   - **Contacts:** `supabase.from('contacts').select('*, contact_organizations(organizations(name))').is('deleted_at', null)` — flatten org names to semicolon-joined string. Columns: id, first_name, last_name, email, phone, title, tags (semicolon-joined), organizations (names semicolon-joined), notes, created_at.
   - **Organizations:** `supabase.from('organizations').select('*').is('deleted_at', null)`. Columns: id, name, type, phone, website, address, city, state, tags (semicolon-joined), notes, created_at.
   - **Deals:** `supabase.from('deals').select('*, pipeline_stages(name), organizations(name)').is('deleted_at', null)`. Columns: id, title, stage (from pipeline_stages.name), value, currency, organization (from organizations.name), expected_close, closed_at, notes, created_at.
5. Implement `escapeCsvField()` per RFC 4180 — quote fields containing comma, newline, or double-quote; double-up internal quotes.
6. Implement `toCSV(rows, columns)` — header row + data rows joined by newline.
7. Prepend UTF-8 BOM (`\uFEFF`) for Excel compatibility (handles Hebrew text per Claude's discretion).
8. Return `new Response(csv, { headers: { 'Content-Type': 'text/csv; charset=utf-8', 'Content-Disposition': 'attachment; filename="contacts-2026-02-23.csv"' } })` with ISO date in filename.
